### 📋 第一周：详细任务清单

**学习目标**：
1.  理解 LangChain 是什么，为什么需要它。
2.  成功配置 Python 虚拟环境并安装 LangChain 及其核心依赖。
3.  配置至少一种大模型API（推荐DeepSeek或OpenAI），并确保可调用。
4.  运行第一个 LangChain 程序，感受从Prompt到模型输出的完整链路。

**具体任务拆解**：

- [ ] **任务 1: 概念建立 (约1小时)**
    - 阅读 LangChain 官方文档首页（[https://www.langchain.com/](https://www.langchain.com/)）和 “Introduction” 部分。
    - 理解核心概念：Model I/O, Retrieval, Chains, Agents, Memory 分别解决什么问题。
    - *思考题*: 对比直接调用API，使用LangChain带来了哪些额外价值？

- [ ] **任务 2: 环境隔离与包安装 (约30分钟)**
    - 在你的项目目录下创建 Python 虚拟环境: `python -m venv venv`
    - 激活虚拟环境。
    - 安装核心包:
      ```bash
      pip install langchain langchain-core langchain-community
      # 根据你要用的模型安装对应包，例如：
      pip install langchain-openai  # 如果用OpenAI
      pip install langchain-deepseek # 如果用DeepSeek (如果官方包未出，先用openai兼容接口)
      pip install python-dotenv    # 用于管理环境变量
      ```

- [ ] **任务 3: API密钥配置与测试 (约30分钟)**
    - 在项目根目录创建 `.env` 文件，并写入你的 API Key。
      ```
      # 如果用DeepSeek
      DEEPSEEK_API_KEY="sk-your-key-here"
      DEEPSEEK_BASE_URL="https://api.deepseek.com/v1"
      ```
    - 创建 `.gitignore` 文件，添加 `venv/` 和 `.env`，防止密钥泄露。
    - 编写一个简单的 `test_api.py` 文件，测试能否直接调用API。

- [ ] **任务 4: 你的第一个 LangChain 程序 (约1小时)**
    - 创建一个 `hello_langchain.py` 文件。
    - 编写代码，完成以下流程：
        1.  加载环境变量。
        2.  初始化一个ChatModel（如 `ChatOpenAI` 或 `ChatDeepSeek`）。
        3.  使用 `ChatPromptTemplate` 创建一个简单的模板。
        4.  使用 LCEL（`|` 操作符）将 Prompt 和 Model 串联成一个 Chain。
        5.  调用 Chain，输入一个简单问题，并打印结果。
    - *验证*: 程序成功运行，控制台输出了模型生成的回答。

- [ ] **任务 5: 问题记录与资产整理 (持续)**
    - 在本地创建一个 `Week1_Learning_Log.md` 文件，记录你遇到的问题、解决方法以及关键概念的理解。
